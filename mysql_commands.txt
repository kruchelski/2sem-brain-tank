## COMANDOS BÁSICOS PARA ACESSAR O MYSQL PELO TERMINAL ##

1) Para acessar o mySQL:
/usr/local/mysql/bin/mysql -uroot -p

2) Vai ser necessário inserir a senha do mysql
É aquela coisa angustiante -> ......G....4f0d4

3) Depois é só utilizar os comandos do banco de dados:
>> show databases
>> use *nome_do_banco_de_dados*
>> show tables
>>...
>>...
